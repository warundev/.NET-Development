I've developed a comprehensive Windows Forms application to streamline hospital operations, featuring robust security and patient management functionalities.

## Key Features
- **Secure Login System**:  Enhanced security measures to protect sensitive patient data.
- **Add Patient**:  Seamlessly add new patients with details like name, index number, age, gender, mobile number, email, and address.
- **View Patient**:  Easily view patient details through an intuitive interface.
- **Fast Response**:  Optimized for quick and responsive interactions.
- **Simple Installation**:  Easy setup and installation process.
- **User-Friendly System UI**:  Designed with a focus on simplicity and ease of use.

## Built with
- .NET Framework
- C#


## Usage
1. **Login**: Use the default credentials to log into the system:
   - **Username:** Admin
   - **Password:** 123
2. **Add Patient**: Navigate to the "Add Patient" section and fill in the patient's details.
3. **View Patient**: Go to the "View Patient" section to see the list of all patients and their details.
